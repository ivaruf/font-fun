font -> base64

Lag en base64 enkodet string av en font

                
                    cat Haymaker.ttf | base64 -w 0 > font64

Brukes rett i css

                
                    @font-face{
                        font-family: hay;
                        src: url(data:font/true-type;base64,AAEAAAASAQAABAAgR1BPU9il5uw...)format('truetype')
                    }
